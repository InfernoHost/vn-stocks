## Currency System

GSC uses a dual currency system for maximum precision:

- **1 Cog = 64 Spurs**
- You start with **10 Cogs** (640 Spurs)
- All prices shown in Cogs (e.g., "15.75 Cogs")
- Internal calculations use Spurs for accuracy

Think of Cogs like dollars and Spurs like cents, but with a 64:1 ratio instead of 100:1.
